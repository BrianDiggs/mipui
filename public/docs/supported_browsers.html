Mipui Browser Support

Mipui officially supports Google Chrome, Mozilla Firefox and Apple Safari. The author uses Chrome almost exclusively, so it's unfortunately likely bugs specific to other browsers might sneak in... if you spot such a bug, please let the author know about it!

This app relies on modern web technologies that are not supported by Microsoft Internet Explorer and Microsoft Edge, so they are not supported and are unlikely to work.
