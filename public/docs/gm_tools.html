Mipui Game Master Tools

When a user loads a Mipui map, that can either be in read/write mode (in which case the address looks like /index.html?mid=something&secret=something) or read-only mode (in which case the address lacks the &secret=something part). GM tools appear differently in the two modes.

The hide / reveal tools will overlay the cell with solid black in read-only mode, and translucent overlay in read/write mode. The basic tool will just show/hide a single cell, but there are buttons to show/hide the entire map, and a tool which reveals all cells in line of sight from the selected cell.

The GM note tool will create text which is invisible in read-only mode.

The hidden secret door tool will create a secret door which is invisible in read-only mode.

The region tool will create a translucent overlay on the cell which is invisible in read-only mode.

GM tools are not cheat-proof. The data is sent from the server, it's just not displayed to the user - so it's still possible to access it. Don't rely on it if you can't trust your viewers to not cheat.
